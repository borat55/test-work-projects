Instagram Clone Coding

My first personal project after learning HTML and CSS. I wanted to make it exactly same as how I see Instagram from my Cellphone. It contains 7 pages; Instagram first page asking to create an account or to login, Login page, Instagram main page, My story, Post content, Activity(which shows “Like” activity), Message.

Coin Tracker

It shows the information of cryptocurrency. If you click a name of the cryptocurrency, the page will show you all about the cryptocurrency, by explaining about the coin, and comparing the price of coin, and visualizing the price per each time or day in line chart/candle stick chart. It fetches data from cryptocurrency API by using React Query, and presents the information in real time in the page. The light and dark mode is built with Styled Components and Recoil. The charts are built with Apex Chart.

Chrome App Clone Coding

Chrome App that has a to-do list, date/time, weather forecast and quote for the day. In the to-do list, there are 3 categories; to-do, doing, done. You can move a to-do to other categories from to-do(default) according to the status of the to-do. For the weather forecast, if you put a name of city correctly, it shows you the weather information of the city. I used React Hook Form to validate forms, and Recoil to change / manage the status of components.
